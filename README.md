<h1 align="center">🚦 Road Accident Data Analysis Dashboard</h1>

<h2>📌 Project Overview</h2>
<p>
This project involves the development of an <b>interactive and dynamic dashboard in Microsoft Excel</b> 
to analyze <b>Road Accident Data</b> for the years <b>2021 and 2022</b>. 
The main objective is to provide the <b>Ministry of Transport stakeholders and emergency services</b> 
with actionable insights into accident severity, casualty trends, and regional factors to help improve road safety.
</p>

<h2>📂 Data Source</h2>
<p>
The dataset used for this project was obtained from 
<b>Kaggle</b>. It is a demo dataset containing approximately 
<b>307,000 rows</b> and was modified for learning and practice purposes. 
The dataset does not represent real-time government or industry data.
</p>

<h2>⚙️ Project Workflow</h2>

<ul>
<li><b>Requirement Gathering:</b> Identifying key KPIs such as Total Casualties, Fatal Casualties, Serious Casualties, and Slight Casualties.</li>

<li><b>Data Cleaning:</b> Handling missing values, correcting formatting issues, and fixing typographical errors to ensure accurate analysis.</li>

<li><b>Data Processing:</b> Creating a custom <b>Month column</b> using Excel formulas to enable time-based trend analysis.</li>

<li><b>Data Analysis & Visualization:</b> Building <b>Pivot Tables and Charts</b> including Donut Charts, Bar Charts, and Tree Maps to analyze casualties by vehicle type, road surface, and location.</li>

<li><b>Dashboard Creation:</b> Designing an interactive Excel dashboard with <b>Timeline and Slicer filters</b> for dynamic data exploration.</li>
</ul>

<h2>✨ Key Features</h2>

<ul>
<li><b>Dynamic Filtering:</b> Filter data by <b>Year, Month, and Area Type (Urban/Rural)</b> using slicers.</li>

<li><b>KPI Tracking:</b> Quick visibility of important metrics such as <b>Total Casualties</b>.</li>

<li><b>Interactive Dashboard Design:</b> Custom-designed filter panel and navigation links.</li>

<li><b>Technical Breakdown Sheet:</b> A separate sheet explaining Pivot Tables and formulas used in the dashboard for better maintainability.</li>
</ul>

<h2>🛠️ Tools Used</h2>

<ul>
<li><b>Microsoft Excel</b> – Data Cleaning, Analysis, Visualization, and Dashboard Creation</li>
<li><b>Kaggle</b> – Dataset Source</li>
</ul>

<h2>📊 Project Outcome</h2>
<p>
The final dashboard provides a clear and interactive view of accident patterns, 
helping users understand casualty distribution by vehicle type, road conditions, 
and geographic factors. This enables better insights into road safety trends.
</p>
